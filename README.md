# アプリケーションの概要
サイコロゲームPig_gameをJavascriptで作りました。
DOMの階層構造の勉強になりました。
[Pig_gameとは](https://en.wikipedia.org/wiki/Pig_(dice_game))

#アプリケーションの機能一覧
- サイコロを振って目を競う対戦機能
- 点数が計上される
- 点数リセット機能
- 勝利の基準点の設定機能
- challenge.js(拡張版)はサイコロが2つに増えます
- challenge.js(拡張版)は勝利条件の点数が設定できます(設定しなければ100点が勝利条件となります)

# アプリケーション内で使用している技術一覧
- 言語 Javascript
